## 7. The compliance factorization (theory)

The paper factors decision compliance into a retrieval term and a use term:

```
P_comply(d) = P_ret(d) · κ(d)
```

Because `∂P_comply/∂P_ret = κ`, under separability `P_comply ≤ κ` — a hard **use
ceiling**. The "use ceiling" names the empirical `κ ≈ 0.6` band that bounds similarity
retrievers on real code (§1); Brief's `κ = 0.703` is the only arm above it. Similarity
recovery decays super-geometrically in causal depth, `P_ret^sim(d) = s_0^d · ρ^{d(d+1)/2}`,
while bounded traversal pays only `q^d`; calibrated at `s_0 = 0.70, ρ = 0.67, q = 0.97`
the structured advantage is `g(1) ≈ 0.50`, `g(2) ≈ 0.79`, with reported crossover
`d⋆ = 2` (closed-form `d⋆ = 1`, the headline `d⋆ = 2` flagged calibrated post-hoc).
