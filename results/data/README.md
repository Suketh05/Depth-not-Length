# results/data — provenance

All files here are transcribed VERBATIM from the paper **"Depth, Not Length: A
Benchmark and Theory of Memory, Retrieval, and Decision Compliance in LLM Coding
Agents"** (the authoritative source of truth). They are the paper's reported results,
NOT regenerated by this repo's offline harness.

Each value can be traced to a labelled table in the manuscript. Nothing in this
directory is "measured" by the code in this repository.

## Files

| file | paper table(s) | what it holds |
|---|---|---|
| `paper_realcode_pooled.csv` | `tab:realret` | Real-code pooled (dcbench+swebench, Claude): recall, compliance, precision, use factor κ per arm |
| `paper_synthetic_compliance.csv` | `tab:synthall` | Synthetic-suite compliance per arm (Claude) |
| `paper_depth_slope.csv` | `tab:slope` | Synthetic compliance depth slope per arm + Brief-minus-best-similarity gap by depth |
| `paper_supersession.csv` | `tab:super` | Supersession: current-decision-ranked-first % (chance = 50%) |
| `paper_public_benchmarks.csv` | `tab:stdbench`, `tab:t101pub` | LoCoMo, DMR, SWE-ContextBench, HotpotQA — Brief vs next-best system |
| `paper_token_economics.csv` | `tab:tok_context_winrate`, `tab:tok_context_by_competitor` | Session-token win rate overall and per competitor |
| `paper_scorecard.csv` | `tab:scorecard` | Unbiased 41-axis scorecard tally (22 W / 14 m / 5 l) |
| `competitor_landscape.csv` | `tab:landscape` | Competitors' own published headline numbers (orientation only; cited) |
| `swe_context_bench.csv` | `tab:t070` | SWE-ContextBench leaderboard (published competitors; Brief row is modeled in the paper) |

## Naming

- `arm` / `system` use the paper's arm names; `brief` = the `brief_graph_3hop` typed
  decision-graph arm ("Brief" in the manuscript).
- Blank cells mean the paper reports no value for that cell (e.g. κ for `none`).

## Notes / gaps

- `paper_synthetic_compliance.csv` has the six arms `tab:synthall` reports (brief, tfidf,
  bm25, dense, raptor, none). The paper's synthetic-compliance table does NOT list
  `hybrid_rrf` or `rerank_ce`, so they are omitted here (their depth values live in
  `paper_depth_slope.csv`, sourced from `tab:slope`).
- SWE-ContextBench appears in two paper tables with different framings: the headline
  unified-harness number (`tab:stdbench`: Brief 47.3, next-best Unabyss 37.6) feeds
  `paper_public_benchmarks.csv`, while the dedicated leaderboard with token cost
  (`tab:t070`: next-best Oiya 38.7) feeds `swe_context_bench.csv`. Brief's SWE-ContextBench
  row is flagged "modeled" in the paper (`tab:t070` caption).
