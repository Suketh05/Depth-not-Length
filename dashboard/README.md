# BriefBench dashboard

Interactive, Python-backed results viewer (Streamlit + Plotly) for the **paper's reported
numbers**. Tabs for real-code retrieval/use, synthetic compliance, depth slope,
supersession, public benchmarks, token economics, the 41-axis scorecard, and the
competitor landscape — each rendered as charts and tables.

Reads the paper-sourced CSVs in [`results/data/`](../results/data) (`paper_*.csv`,
`competitor_landscape.csv`, `swe_context_bench.csv`), which are transcribed verbatim from
the paper *Depth, Not Length* (the source of truth). Nothing here is regenerated by this
repo's offline harness.

## Run locally
```bash
pip install -r dashboard/requirements.txt
streamlit run dashboard/app.py
```

## Deploy a public URL (free, for marketing)
1. Push this repo to GitHub.
2. Go to https://share.streamlit.io → New app → point at `dashboard/app.py`.
3. Share the URL.

Honest by design: Brief leads on its design axes but loses 5 of 41 scorecard axes
(e.g. HotpotQA supporting-fact recall to Zep) — shown straight.
